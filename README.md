# 🇲🇦 DarijaInsight

**DarijaInsight** is an AI-powered platform designed to help Moroccan businesses understand customer opinions written in Moroccan Darija.

The project combines **Natural Language Processing (NLP)** and **Deep Learning** to automatically classify customer reviews into **Positive**, **Neutral**, or **Negative** sentiments. Built on a fine-tuned MARBERT model, the platform enables companies to analyze large volumes of customer feedback in real time and make data-driven decisions through an intuitive dashboard.

---

# 🎯 Project Objectives

The main objectives of this project are to:

- Analyze customer reviews written in Moroccan Darija.
- Automatically classify sentiments into **Positive**, **Neutral**, and **Negative**.
- Provide confidence scores for every prediction.
- Offer an interactive dashboard for review management and visualization.
- Help businesses better understand customer satisfaction and improve decision-making.
